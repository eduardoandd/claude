Context Caching — OpenAI

O Context Caching armazena partes repetidas do prompt — como instruções do sistema e dados cadastrais da empresa — e as reutiliza a 10% do custo original.

Quanto mais longa a conversa, maior o aproveitamento do cache. No trace 4 (Verificação de Audiências), 67% dos tokens de entrada foram servidos do cache — atingindo a máxima eficiência do ciclo.

Isso torna conversas longas e complexas economicamente viáveis para aplicações jurídicas, onde contexto acumulado é essencial.
